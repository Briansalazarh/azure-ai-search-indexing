# Mineração de Conhecimento e IA Search no Azure

Este repositório documenta a aplicação prática de técnicas de organização e pesquisa de documentos utilizando o Azure AI Search. O foco principal é a ingestão de dados, indexação e enriquecimento de conteúdo através de Inteligência Artificial.

## Objetivos e Práticas
O projeto foi desenvolvido seguindo três etapas fundamentais de uma solução de pesquisa cognitiva:

1. **Ingestão de Dados**:
   - Configuração de fontes de dados para o Azure AI Search.
   - Importação de documentos não estruturados e dados brutos.

2. **Criação de Índices e Enriquecimento com IA**:
   - Definição de índices de pesquisa.
   - Aplicação de "skills" cognitivas (AI enrichment) para extrair informações relevantes dos dados (OCR, detecção de linguagem, extração de entidades).

3. **Exploração de Dados**:
   - Realização de consultas (queries) nos dados indexados.
   - Validação dos resultados obtidos através da pesquisa cognitiva.

## Conclusões
A prática demonstrou como ferramentas de pesquisa cognitiva podem transformar grandes volumes de dados não estruturados em conhecimento pesquisável e organizado, facilitando a tomada de decisão e a análise de informações.

# Minería de Conocimiento y IA Search en Azure

El proyecto fue desarrollado siguiendo tres etapas fundamentales de una solución de búsqueda cognitiva:

   - Configuración de fuentes de datos para Azure AI Search.
   - Importación de documentos no estructurados y datos brutos.

2. **Creación de Índices y Enriquecimiento con IA**:
   - Definición de índices de búsqueda.
   - Aplicación de "skills" cognitivas (AI enrichment) para extraer información relevante de los datos (OCR, detección de idioma, extracción de entidades).

3. **Exploración de Datos**:

La práctica demostró cómo las herramientas de búsqueda cognitiva pueden transformar grandes volúmenes de datos no estructurados en conocimiento buscable y organizado, facilitando la toma de decisiones y el análisis de información.
